Задача 1
Есть список чисел, из которого надо удалить элементы, удовлетворяющие определенное условие.Будем удалять из списка, состоящего из 20 чисел в диапазоне от 0 до 100, все элементы, которые больше 35 и меньше 65. При этом удаляемые числа сохраним в другом списке.
Примеры выполнения программы:
A = [80, 35, 30, 98, 35, 4, 94, 51, 22, 22, 52, 97, 67, 90, 9, 1, 87, 78, 100, 29] - начальнй список
A = [80, 35, 30, 98, 35, 4, 94, 22, 22, 97, 67, 90, 9, 1, 87, 78, 100, 29] - отфильтрованый список
B = [51, 52] - список с удаленным елементами
